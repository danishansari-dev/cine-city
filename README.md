# CineCity

A 3D city where every building is a film

## Setup

1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env.local` and add your API keys (including `NEXT_PUBLIC_TMDB_API_KEY`)
4. Run `npm run dev`
